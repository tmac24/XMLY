##基本运算符
- 赋值运算符 

```
let b = 10
var a = 5
a = b
// a 现在等于 10
```

- 算术运算符: 加减乘除；求余运算符
- 组合赋值运算符
- 比较运算符
- 三元运算符：a ? b : c
- 区间运算符

```
for index in 1...5 {
    print("\(index) * 5 = \(index * 5)")
}
// 1 * 5 = 5
// 2 * 5 = 10
// 3 * 5 = 15
// 4 * 5 = 20
// 5 * 5 = 25
```
- 逻辑与运算符

逻辑与运算符（a && b）表达了只有 a 和 b 的值都为 true 时，整个表达式的值才会是 true。
